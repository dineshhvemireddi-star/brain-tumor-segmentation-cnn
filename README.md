# Brain Tumor Segmentation and Classification using CNN

This academic project demonstrates brain tumor detection using MRI images and Convolutional Neural Networks (CNN).

The system performs image preprocessing, feature extraction, model training, segmentation, and classification of tumor images.

Technologies Used:
- Python
- CNN (Deep Learning)
- Image Processing
- TensorFlow / Keras
